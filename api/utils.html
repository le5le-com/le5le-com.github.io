Utils #

formatPadding #

将 padding 转换成数组格式 [top, right, bottom, left]
padding 规则与 css padding 相同

参数：

  • padding: Padding
type Padding = number | string | number[];

返回：
number[]

示例：

formatPadding(5)
// [5, 5, 5, 5]
formatPadding('5')
// [5, 5, 5, 5]
formatPadding([3, 5])
// [3, 5, 3, 5]
formatPadding([3, 4, 5])
// [3, 4, 5, 4]
formatPadding([3, 4, 5, 6])
// [3, 4, 5, 6]

rgba #

生成具有透明度的颜色，通常用于生成当前色的浅色

参数：

  • c: string
    不带透明度的十六进制颜色，如：#1890ff
  • p: number
    透明度，0-1

返回：
string

示例：

rgba('#f40', 0.1)
// 'rgba(255,68,0,0.1)'
