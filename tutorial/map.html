鹰眼地图 #

鹰眼地图是指在画布右下角弹出一个缩略图，鼠标点击可以跳到指定位置。

在线体验： http://t.le5le.com/

示例：

// 显示鹰眼地图
topology.showMap();

// 关闭鹰眼地图
topology.hideMap();
