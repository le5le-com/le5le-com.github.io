乐吾乐 2D 可视化组态软件 #

【乐吾乐 2D 可视化组态软件】是乐吾乐科技完全自主研发、集动态数据显示和事件交互于一体的可视化工具，把复杂、变化、多样、有价值的大数据变成直观易懂、实时显示的可视化交互显示，具有跨平台、实时监控、动态交互、高效、可扩展、支持自动算法等特点，最大程度减少研发和运维的成本，并致力于普通业务人员 0 代码开发实现 web 组态、SCADA 等场景。是拥有为电力能源、水利、物联网、工业互联网、智慧城市、智慧医疗、智慧农业、IT 运维等提供解决方案的可视化平台。

特色 #

丰富的组态能力

  • 拖拽编辑，简单易用。支持基础图形、UML图形、物联网图形、国家标准电力图形、大屏、图表、按钮、视频监控等组件，支持自定义图形库扩展能力
  • 快速数据绑定，实时数据动态显示，数据下发，双向数据通信，同时支持定制中间件条件格式化或条件筛选显示
  • 丰富的动画功能，配合条件很方便的进行动画告警，并支持节点逐帧自定义动画
  • 丰富的事件、自定义消息，支持组态联动，通过事件或消息发送设备指令
  • 支持组件多样化的外观属性设置，支持组合组件，拥有格式刷功能
  • 支持弹窗功能、开关状态切换、条件触发器、实时视频监控功能等

便捷的数据对接能力，0代码完成数据通信

  • 支持 mqtt 动态数据监听
  • 支持 websocket 动态数据监听
  • 支持 http 自主请求动态更新数据
  • 支持静态数据创建，导入

组态的应用

  • 支持组态图的导入导出及组态图源代码导入导出
  • 支持组态图环境迁移能力
  • 支持通过API对组态进行创建、查询、删除、复制、发布等管理能力
  • 支持组态设计页面内嵌集成到第三方平台
  • 支持组态查看页面内嵌集成到第三方平台
  • 提供离线部署安装方式，支持云端部署

多端适配能力

  • 支持Chrome、Firefox、Edge等主流浏览器。
  • 移动端支持webview方式访问

强大的系统扩展能力

  • 图形库扩展：支持各种自定义图形库扩展，格式支持js、svg、jpg、iconfront
  • 数据显示扩展：支持自定义中间价，对数据格式进行转换并显示
  • 交互事件扩展：支持自定义消息、生命周期hook等，方便定制扩展各种业务中间件
  • 智能算法扩展：支持自定义拖拽智能算法，自定义连线算法等
  • 动画扩展：支持节点逐帧定义自定义动画；支持自定义图形库自定义算法动画
  • 排版布局扩展：支持自定义排版布局算法

追求卓越性能

  • 产品设计简单实用，具有良好的可视化和界面友好度，系统交互中断不影响系统运行
  • 支持10000-20000节点
  • 支持 1000+动画播放，大数据、高要求场景无压力
  • 1000数据点30ms完成刷新

海量的组件库资源

  • 拥有电力系统，能源系统，物联网，智慧水务，智能制造，数字大屏，图表控件，视频流监控一共4000多个组件。拖拽编辑，简单易用

丰富的应用场景案例

  • 架构图、拓扑图、UML图、脑图等
  • 电力能源、水利水务、变电站、光伏系统、火电厂、风电、矿山、煤矿系统等
  • 物联网、工业互联网、电信系统、数据中心、采暖系统、远程监控系统等
  • 智慧工厂、智慧楼宇、智慧园区、智慧交通、智慧城市、智慧停车、智慧医疗、智慧农业等
  • 大屏展示、看板展示、数据报表、安防监控、IT 运维等

国产开源 自主可控

  • 乐吾乐公司自主潜心研发三年，掌握核心技术和知识产权，能够做到完全自主可控，且产品保持持续迭代优化
