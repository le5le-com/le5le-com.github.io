表格 #

table 类型控件

  • 数据

  1. table 字段

    名称类型描述
    rowHightnumber行高，默认 30
    colWidthnumber列宽，默认 100
    headerobject表头配置，见下方header配置。
    dataobject[]|array数据配置。1.当类型为objec[]时，见下方data配置；2.当数据类型为array时，参见下方data配置中的data属性。

  2. header配置

    名称类型描述
    heightnumber表头高度
    showboolean是否显示
    fontWeightstring文字加粗
    dataobject[]表头单元格数据样式配置。
    {
      text:string,//列名
      width:number//列宽
    }[]

  3. data配置

    名称类型描述
    heightnumber行高
    data(array|string|object)[]行数据。1.当类型为string时，显示纯文本;2.当为object时，可以配置单元格样式(背景、文字和线条颜色)；3.当为array时，可以添加子节点，子节点的配置可参考Pen

注意：1.子节点的分布：padding默认为10，当子节点布局超出单元格宽度时会换行，当子节点布局高度小于单元格高度时会居中对齐。

  • 数据更新

1.1.16版本以后，我们提供了一个规则来更新一个单元格。使用方式如下：

topology.setValue({
    id:'table.id',//表格id
    row:1,//第几行
    col:1,//第几列
    value:200 //新值
  });
