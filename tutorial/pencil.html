铅笔 #

铅笔是可以任意涂鸦的绘图小工具

在线体验： http://t.le5le.com/

示例：

// 开始铅笔绘画
topology.drawingPencil();

// 鼠标抬起结束

// 停止铅笔绘画（关闭铅笔绘画状态）
topology.stopPencil();
