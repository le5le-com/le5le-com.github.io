# 常见问题

## 是否可以商用

基于开源协议 MIT 核心库自己开发的可以商用；

试用企业版非 MIT 项目，需要购买授权；

其他开发者作品参考开发者作品协议；

## 如何编译核心库源码

1. 下载核心库源码

```shell
git clone https://github.com/le5le-com/topology.js.git
```

2. 安装依赖包

```shell
// 使用yarn。我们使用的是yarn workspace目录结构
yarn
```

3. 编译(Build)

```shell
yarn build
// 编译后的文件在dist目录下
```
