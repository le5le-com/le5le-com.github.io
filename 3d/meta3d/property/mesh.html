模型 #

模型包含的TargetType类型如下：

  • TargetType.TransformNode
  • TargetType.AbstractMesh
  • TargetType.Mesh
  • TargetType.LinesMesh

虽然都属于模型的属性，但是可以应用的TargetType可能也不同，使用时请注意区分。

enabled #

是否启用。

类型

ValueType.Boolean

可以通过事件/自定义数据/动态数据修改

true

可以设置动画

false

目标类型

  • TargetType.TransformNode
  • TargetType.AbstractMesh
  • TargetType.Mesh
  • TargetType.LinesMesh

visibility #

透明度。取值范围0 - 1，0表示透明，1表示不透明。

类型
